This website is in development mode. I'm writing a custom web server in C to support dynamic server-side-rendering for my blog. My tech stack is Html + CSS + JS + C. Right now I'm using BULMA CSS to style this page, but I wish to learn something more flexible and powerful like Tailwind CSS. I don't have good exposure to web development, and I'm building this website and server together iteratively, so things might look sad and bad.

Since the webserver is written in C, it's possibly vulnerable to some exploitation techniques. While I'm very strict with my error checking, I can never be sure there exists any vulnerabilities or not. Wanna try? go on! Just don't do anything that will hurt me to get back this website up. If you find something cool, ping me at admin[at]brightprogrammer[dot]in and if I find it interesting, I'll share your findings on my blog post and attribute you :-)
